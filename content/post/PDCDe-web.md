Hola, hoy continuaré con una de las sagas más interesantes del *blogo*: **Proceso de creación de...** Hoy será de una página web

***1- Saber qué quieres hacer***

Tienes que tener una idea de lo que quieres crear

***2- Crear un repositorio***

Puede ser en GitHub, GitLab,... pero tienes que crearlo
>Un repositorio contiene todos los archivos de tu proyecto y el historial de revisiones de cada uno de ellos. Puedes debatir y administrar el trabajo de tu proyecto dentro del repositorio.

***3- Programar***

![Cat](https://media.tenor.com/t3yHhG6m4YkAAAAC/cat-work.gif)

***4- Subir la web a Internet***

Tienes que usar una web llamada *Netlify*. Tienes que conectarla a GitHub/Lab y luego crear un nuevo *site*. Luego seleccionas el repositorio de la web que has creado antes. Posteriormente le cambias el nombre al *site* para customizarlo y si tienes un dominio (.es, .tk, .com,...) configurarlo.

Y ya está, es una explicación muuuuy básica, pero porque es mejor ver un tutorial. Este post es para darte una idea de cómo se hace.
